# WordZoo Design Context

## Product
Language learning SaaS — AI-generated keyword mnemonics and spaced repetition for vocabulary acquisition. Currently focused on Indonesian for English speakers.

## Audience
Adult beginners (25-45) learning Indonesian — travelers, expats, language hobbyists. Not academic linguists. They want to feel progress, not be overwhelmed. Many use the app on mobile during commutes or before bed.

## Brand Personality
**Warm, approachable, slightly whimsical.** The "Zoo" in WordZoo is playful but not childish. Think: a well-designed cookbook meets a pocket field guide. Friendly without being patronizing. Calm during active learning — the UI should disappear when users are focused.

## Design Principles
1. **Warm over sterile** — tinted surfaces, rounded forms, soft shadows. Never cold SaaS gray.
2. **Focus over flash** — during learning (review cards, scene flow), the content is the hero. UI fades back.
3. **Calm confidence** — no urgency, no guilt. "5 items ready" not "5 items overdue!" Progress is celebrated gently.
4. **Legibility first** — Indonesian words displayed large and clear. Mnemonics readable at a glance. Body text comfortable for extended reading.

## Key Use Cases (by frequency)
1. **Review session** (daily) — flip cards, rate recall, 2-5 minutes
2. **Learn new scene** (2-3x/week) — intro, dialogue, phrases, vocab, patterns, summary
3. **Dashboard check** (daily) — see what's due, check streak, continue learning
4. **Browse paths** (weekly) — see progress through learning path

## Anti-patterns (banned)
- Glassmorphism / backdrop-blur (reads as "AI slop")
- Pure white (#FFFFFF) or pure black (#000000) backgrounds
- Bounce/elastic easing on buttons
- Infinite pulse/glow animations
- More than 3 accent colors competing for attention
- Step numbers on dashboard sections ("Step 1: Review")

## Color Intent
- **Primary accent (action)**: Blue — buttons, links, interactive elements
- **Language accent (Indonesian)**: Orange — foreign words, language-specific content
- **Neutrals**: Warm-tinted, not blue-gray
- **Semantic**: Success (green), error (red), warning (amber) — used sparingly

## Typography Intent
- **Display/headings**: Nunito — warm, rounded, friendly
- **Body**: Nunito — excellent Latin Extended support for Indonesian
- **Mono**: Geist Mono — code/technical contexts only
- **Scale**: 1.25 ratio modular scale from 16px base
